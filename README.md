# Castlevania_Bader
Practica Remake 2D

Soc Bader i el plan es recrear el primer nivell del primer castlevania, a priori nomes l'entrada del castell de Lord Dràcula, y potser una mica més de la següent habitació.

El protagonista del primer joc es Simon Belmont, pero potser es cambia per Soma Cruz o Alucard (Adrian Fahrenheit Tepes).

El moviment serà basic, dreta, esquerra, salt, un atac horitzontal.

Els enemics seran els mateixos que en el joc original.

I es mantendrán els objectes destruibles.

Tot susceptible als canvis que siguin convenients.
